# Dev

1. Clone the project.
2. Copy ```.env.template``` and rename to ```.env```.
3. Execute
```
yarn install
```
4. Start image (Docker Desktop needed)
```
docker-compose up -d
```

5. Start the Nest backend
```
yarn start:dev
```
6. Open site ```localhost:3000/graphql```
